# Projet ERPNext v13

Ce projet contient une installation Frappe/ERPNext v13 avec les apps suivantes :
- ERPNext
- gestion_formation

## Installation

1. Cloner le projet
2. `cd frappe-bench-ofs`
3. `bench start`

## Configuration

Les fichiers de configuration sensibles ne sont pas inclus pour des raisons de sécurité.
